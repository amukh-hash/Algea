# Accessibility keyboard interactions

## Command palette
- Open/close: `Ctrl/Cmd+K`.
- Navigate entries: Arrow Up/Down.
- Close: Escape.

## Research table selection
- Focus rows with Tab.
- Toggle selection: Space.
- Open detail: Enter.
- Shift+ArrowDown extends quick range selection.

## Run detail tabs
- Arrow left/right moves focus across tabs.
- Enter/Space activates focused tab (manual activation).
- Tab enters active panel content.

## Compare legend toggles
- Legend rows are keyboard focusable checkboxes.
- Space toggles series visibility.
