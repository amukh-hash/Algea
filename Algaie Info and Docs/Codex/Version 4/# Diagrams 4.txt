## Diagram 1 — High-level component architecture
```mermaid
flowchart LR
  UI[Frontend Next.js UI] --> API[FastAPI API backend.app.api.main]

  DATA --> MODELS[Model Training/Eval]
  MODELS --> ART[Run Artifacts / production_pack]
```

## Diagram 3 — Control flow / orchestration
```mermaid
sequenceDiagram
  participant U as User/CLI
  participant E as Entrypoint
  participant C as Config Loader
  participant D as Data+Features
  participant M as Model
  participant R as Risk/Constraints
  participant X as Execution
  participant T as Telemetry
  U->>E: start run
  E->>C: load config(s)
  C-->>E: resolved settings
  E->>D: build universe/features
  D-->>E: feature frame
  E->>M: generate decisions
  M-->>E: signals/scores
  E->>R: apply constraints
  R-->>E: target orders
  E->>X: send broker/paper orders
  X-->>E: fills/status
  E->>T: emit metrics/events/artifacts
```
