# Accessibility Keyboard Guide

## Command palette
- `Ctrl/Cmd + K` opens/closes palette.
- `Esc` closes palette.
- `Enter` activates selected command.

## Research table
- Tab to header controls and row checkboxes.
- `Space` toggles checkbox.
- Use linked run name to open detail.

## Run detail tabs
- Arrow keys move focus between tabs.
- `Enter` / `Space` activates focused tab.
- Tab moves into active panel.

## Compare legend/toggles
- Tabbable toggles for alignment and normalize controls.
- Metric dropdown keyboard-selectable with native select semantics.
